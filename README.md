# 🍔 GourmetOn

Landing page moderna desenvolvida para apresentar o aplicativo fictício de delivery GourmetOn.

O projeto foi criado utilizando React, Vite e Tailwind CSS, com integração de API externa para exibir pratos dinamicamente através de requisições assíncronas com Fetch API.

---

# 🚀 Tecnologias Utilizadas

* React
* Vite
* Tailwind CSS
* JavaScript
* Fetch API
* TheMealDB API

# ⚙️ Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/gourmeton.git
```

Acesse a pasta:

```bash
cd gourmeton
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
